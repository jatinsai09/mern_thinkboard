🧠 MERN ThinkBoard

A full-stack ThinkBoard application built using the MERN Stack — MongoDB, Express.js, React, and Node.js.

ThinkBoard allows users to create, manage, and organize notes efficiently with a clean UI and secure backend architecture.

🔗 Live Demo:
https://mern-thinkboard-wyda.onrender.com

🚀 Features
📝 Core Functionality

✅ Create Notes

✅ View All Notes

✅ Edit Existing Notes

✅ Delete Notes

✅ Full CRUD Operations

✅ Real-time UI updates

✅ RESTful API architecture
